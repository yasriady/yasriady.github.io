Modul Praktikum Teknologi Internet of Things

ESP32 dan Sensor DHT22 Mengirim Data ke Web API

Penulis

Dedy Yasriady, M.Kom.

Diterbitkan

25 Mei 2026

Pendahuluan

Deskripsi Praktikum

Pada praktikum ini mahasiswa akan mempelajari implementasi dasar sistem Internet of Things (IoT) menggunakan:

  • ESP32
  • Sensor DHT22
  • WiFi
  • REST API berbasis HTTP/HTTPS
  • JSON

Mahasiswa akan melakukan pembacaan data suhu dan kelembaban dari sensor DHT22, kemudian mengirimkan data tersebut ke server web menggunakan metode HTTP POST.

Capaian Pembelajaran

Setelah menyelesaikan praktikum ini mahasiswa diharapkan mampu:

  1. Memahami konsep dasar Internet of Things.
  2. Menghubungkan ESP32 ke jaringan WiFi.
  3. Membaca data sensor DHT22.
  4. Mengirim data ke Web API menggunakan HTTP POST.
  5. Memahami format data JSON pada sistem IoT.
  6. Melakukan simulasi IoT menggunakan Wokwi.

Dasar Teori

Internet of Things (IoT)

Internet of Things merupakan konsep dimana perangkat fisik dapat saling terhubung melalui jaringan internet untuk mengirim dan menerima data secara otomatis.

Contoh implementasi IoT:

  • Monitoring suhu ruangan
  • Smart home
  • Smart city
  • Monitoring lingkungan
  • Sistem pertanian pintar

ESP32

ESP32 adalah mikrokontroler yang memiliki fitur:

  • WiFi
  • Bluetooth
  • GPIO
  • ADC
  • PWM
  • Deep Sleep

ESP32 sangat populer dalam pengembangan IoT karena murah dan memiliki kemampuan jaringan yang baik.

Sensor DHT22

Sensor DHT22 digunakan untuk membaca:

  • Suhu
  • Kelembaban udara

Karakteristik DHT22:

Parameter Nilai
Tegangan kerja 3.3V – 5V
Rentang suhu -40°C s/d 80°C
Rentang kelembaban 0% – 100%
Akurasi suhu ±0.5°C

HTTP POST

HTTP POST digunakan untuk mengirim data dari perangkat IoT menuju server.

Pada praktikum ini data dikirim dalam format JSON.

Contoh:

{
  "temperature": 28.5,
  "humidity": 70.2,
  "device_id": "esp32-wokwi-02"
}

Source Code Program

#include <WiFi.h>
#include <WiFiClientSecure.h>
#include <HTTPClient.h>
#include <DHT.h>

// ================= CONFIG =================
const char* WIFI_SSID     = "Wokwi-GUEST";
const char* WIFI_PASSWORD = "";

const char* API_URL   = "https://iotweb.apache.web.id/api/save-data";
const char* API_TOKEN = "YOUR_TOKEN";

const char* DEVICE_ID = "esp32-wokwi-02";

// ===== DHT22 CONFIG =====
#define DHTPIN 15
#define DHTTYPE DHT22

DHT dht(DHTPIN, DHTTYPE);
// ==========================================

void setup() {
  Serial.begin(115200);

  dht.begin();

  WiFi.begin(WIFI_SSID, WIFI_PASSWORD);

  Serial.print("Menghubungkan WiFi");

  while (WiFi.status() != WL_CONNECTED) {
    delay(500);
    Serial.print(".");
  }

  Serial.println();
  Serial.println("WiFi Connected");
}

void loop() {

  float temp = dht.readTemperature();
  float hum  = dht.readHumidity();

  if (isnan(temp) || isnan(hum)) {
    Serial.println("Gagal membaca sensor DHT22");
    delay(2000);
    return;
  }

  Serial.print("Suhu: ");
  Serial.print(temp);
  Serial.println(" °C");

  Serial.print("Kelembaban: ");
  Serial.print(hum);
  Serial.println(" %");

  String json =
    "{"
    "\"temperature\":" + String(temp, 2) + ","
    "\"humidity\":" + String(hum, 2) + ","
    "\"device_id\":\"" + String(DEVICE_ID) + "\""
    "}";

  WiFiClientSecure client;
  client.setInsecure();

  HTTPClient http;

  http.begin(client, API_URL);

  http.addHeader("Content-Type", "application/json");
  http.addHeader("Authorization", String("Bearer ") + API_TOKEN);

  int httpCode = http.POST(json);

  Serial.print("HTTP Code: ");
  Serial.println(httpCode);

  if (httpCode > 0) {
    Serial.println(http.getString());
  } else {
    Serial.println("Gagal mengirim data");
  }

  http.end();

  delay(3000);
}
