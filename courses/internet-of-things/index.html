Notifikasi NodeMCU menggunakn WhatsApp

Author

Dedy Yasriady, S.Kom., M.Kom.

🎯 Tujuan Pembelajaran

Setelah mengikuti praktikum ini, mahasiswa diharapkan mampu:

  • Menghubungkan NodeMCU ke jaringan WiFi
  • Memahami konsep input digital (GPIO)
  • Mengirim data ke server menggunakan HTTP POST
  • Mengirim notifikasi WhatsApp menggunakan API (Sendora)
  • Mengimplementasikan sistem IoT sederhana berbasis trigger

🧰 Alat dan Bahan

  • NodeMCU (ESP8266)
  • Kabel USB
  • Breadboard
  • Push button
  • Resistor (opsional)
  • Koneksi WiFi
  • Akun Sendora (WhatsApp API)

🧠 Konsep Sistem

Sistem ini bekerja dengan konsep event-driven:

  • Tombol ditekan → GPIO bernilai LOW
  • NodeMCU membaca kondisi tersebut
  • NodeMCU mengirim request ke API WhatsApp
  • Pesan diterima oleh pengguna

🔌 Rangkaian Sederhana

  • Salah satu kaki tombol → GPIO4 (D2)
  • Kaki lainnya → GND
  • Gunakan mode INPUT_PULLUP

🔧 Persiapan

1. Koneksi WiFi

Siapkan:

  • SSID
  • Password

2. Konfigurasi Sendora

Siapkan:

  • API Key
  • Device ID
  • Nomor WhatsApp tujuan

💻 Program NodeMCU

#include <ESP8266WiFi.h>
#include <ESP8266HTTPClient.h>
#include <WiFiClientSecure.h>

// Konfigurasi WiFi
const char* ssid = "NAMA_WIFI";
const char* password = "PASSWORD_WIFI";

// Konfigurasi Sendora
String sendora_url = "https://sendora.web.id/api/v1/messages/send";
String api_key = "API_KEY_ANDA";
String device_id = "DEVICE_ID_ANDA";
String no_tujuan = "628xxxxxxxxxx";

#define BUTTON_PIN D2

// =========================
// FUNCTION KIRIM WHATSAPP
// =========================
void kirimWhatsApp(String pesan) {
  if (WiFi.status() == WL_CONNECTED) {
    WiFiClient client;
    HTTPClient http;

    http.begin(client, sendora_url);
    http.addHeader("Content-Type", "application/json");
    http.addHeader("Authorization", api_key);

    String payload = "{";
    payload += "\"device_id\":\"" + device_id + "\",";
    payload += "\"number\":\"" + no_tujuan + "\",";
    payload += "\"message\":\"" + pesan + "\"";
    payload += "}";

    int httpResponseCode = http.POST(payload);

    Serial.print("Response: ");
    Serial.println(httpResponseCode);

    http.end();
  } else {
    Serial.println("WiFi tidak terhubung!");
  }
}

void setup() {
  Serial.begin(115200);

  pinMode(BUTTON_PIN, INPUT_PULLUP);

  WiFi.begin(ssid, password);
  Serial.print("Menghubungkan ke WiFi");

  while (WiFi.status() != WL_CONNECTED) {
    delay(500);
    Serial.print(".");
  }

  Serial.println("
WiFi terhubung!");
}

void loop() {
  if (digitalRead(BUTTON_PIN) == LOW) {
    Serial.println("Tombol ditekan! Mengirim WhatsApp...");

    kirimWhatsApp("Halo! Ini notifikasi dari NodeMCU");

    delay(3000);
  }
}

🔍 Penjelasan Program

Library

  • ESP8266WiFi → koneksi WiFi
  • HTTPClient → komunikasi HTTP
  • WiFiClient → koneksi jaringan

Logika Program

  1. NodeMCU terkoneksi ke WiFi
  2. Tombol ditekan (LOW)
  3. NodeMCU mengirim HTTP POST
  4. API Sendora memproses
  5. Pesan WhatsApp terkirim

🧪 Pengujian

Langkah:

  1. Upload program ke NodeMCU
  2. Buka Serial Monitor
  3. Tekan tombol
  4. Pastikan muncul response
  5. Cek WhatsApp tujuan

⚠️ Troubleshooting

  • WiFi tidak terhubung → cek SSID & password
  • Tidak ada respon → cek API Key
  • Pesan tidak terkirim → cek device_id & nomor
  • Serial error → cek baudrate (115200)

🎓 Tugas Mahasiswa

  1. Modifikasi isi pesan WhatsApp
  2. Tambahkan delay berbeda
  3. Tambahkan LED indikator saat tombol ditekan

💡 Kesimpulan

Praktikum ini menunjukkan implementasi sederhana IoT:

  • Input (tombol)
  • Proses (NodeMCU)
  • Output (WhatsApp)

Mahasiswa diharapkan memahami dasar integrasi hardware dan API.
